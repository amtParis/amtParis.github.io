Course Description

This course will provide an introduction to the fundamentals of front-end web development. Students will learn how to build web pages using HTML, apply styles, layouts and typography with CSS and add interactivity with Javascript. Design strategies and tools such as responsive layouts, web graphics, usability, and accessibility will also be introduced. Students will develop a web-based project by the end of the course.

CSS transition, hover, transform review. Responsive design strategies. Media queries.
