Course Description

This course will provide an introduction to the fundamentals of front-end web development. Students will learn how to build web pages using HTML, apply styles, layouts and typography with CSS and add interactivity with Javascript. Design strategies and tools such as responsive layouts, web graphics, usability, and accessibility will also be introduced. Students will develop a web-based project by the end of the course.

Week 1: Fundamentals

Day 1: Class Intro and HTML Basics

Overview of class, introductions, the internet. Creating html docs : elements, tags, semantic mark-up.

Homework: Create a biography page for yourself (it can be real or invented) using only HTML (no styling) with attention to semantic mark-up. Make a sketch using only boxes that maps out the parts of the page before you begin.

Week 2: Responsive Design, Animation, Javascript

Week 3: Frameworks, Templates, Project Development
